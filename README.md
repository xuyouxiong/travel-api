# SpringBoot_Travel

#### 介绍
基于SpringBoot+MySQL+MyBatis-Plus+VUE仿照马蜂窝的旅游项目，前后端分离的多模块开发项目，使用Redis、MongDB、Elasticsearch等技术

#### 软件架构
SpringBoot的前后端分离项目
